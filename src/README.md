Recuperer les fichiers :

`wget https://www.gutenberg.org/files/345/345-0.txt`
`mv 345-0.txt dracula.txt`

compression :
`tar -cvf [NOM ARCHIVE] [FICHIER1 FICHIER2 ...]`

decompression:
`tar -xvf [NOM ARCHIVE]`
